よくある質問

これは何ですか？
Google Chrome用の拡張ツールです。
何ができますか？
Youtubeのビデオのリストをテキストファイルに保存できます。
どうやって使いますか？
YoutubeのTOPページ、もしくは任意のチャンネルのビデオタブでアイコンを押してください。
チャンネル全ての動画データが取得できないのですが？
あくまであなたのブラウザで表示されているリストになるので、沢山リストがあるチャンネルは頑張ってスクロールしてください。
動画の長さがおかしいです
拡張のアイコンを押した時にブラウザ上に時間表示がされていないと思います。時間の表示は少し遅れて表示されるのでちょっと待ってからクリックしてください。
アップロードした日時を取得したいのですが…
表示上のデータを取得しているため、日付が「1年前」など丸められたデータになってしまうので今回は見送りました。
本当にどこにも情報を送っていませんか？
はい、作者自体もそんな拡張嫌なのでAPIサーバーなど含めてどこにも送っていません。完全にあなたのブラウザの情報をまとめているだけです。

FAQ

What is this?
This is an Extension tool for Google Chrome.
What can this tool do?
You can save the list of Youtube videos to a text file.
How do I use it?
Click the icon when you see the TOP page of Youtube or the video tab of any channel.
Why can't I get all the video data for a channel?
This can download only the list displayed in your browser, so if a channel has many listings, you should scroll through them(many times).
The time(length) of the video is wrong.
The time display will appear with a slight delay, so please wait a moment before clicking on it.
How can I get the date and time of the upload?
The data displayed would be rounded data such as "one year ago" and so on, so it was not used this time.
Are you sure you are not sending information anywhere?
Of course, This extension does NOT send your data anywhere, including API servers, because I do NOT LIKE such extensions too. Instead, it is put together of your browser's data.
